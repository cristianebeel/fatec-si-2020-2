O que é HTML

HyperText Markdown Language é uma linguagem de marcação utilizada na criação de páginas web.

Sua estrutura consiste em tags, que possibilitam ao desenvolvedor criar seções, títulos, links, parágrafos e tabelas além de inserção de componentes como imagens e vídeos. É uma linguagem que permite ao desenvolvedor publicar conteúdos na internet.

Para uma melhor visualização da página, é utilizada em conjunto com CSS e JavaScript.
