O que já vimos em Introdução à Programação

Em Introdução à Programação é utilizada a linguagem JavaScript. Nessa disciplina, nós aprendemos a:

  • Converter string para integer
  • Realizar operações matemáticas básicas (soma, subtração, multiplicação e divisão)
  • Como acionar o click do botão em um formulário
  • Inserir no documento HTML um texto digitado pelo usuário, através de janela prompt ou de input no formulário
  • Como limpar input de um formulário
  • Sortear um número de 1 até um número definido pelo usuário
  • Como ligar um arquivo JavaScript externo em um documento HTML
